# MartinaECC 测试宇宙

这是一个部署在 GitHub Pages 上的测试集合网站。

## 功能

- 首页展示多个测试入口
- `pdp/`：PDP 性格测试，包含 30 道题和 5 种结果类型
- `bfti/`：BFTI 暴富TI测试占位页
- 支持电脑和手机访问

## 目录结构

```text
.
├── index.html
├── style.css
├── pdp/
│   ├── index.html
│   ├── questions.js
│   └── script.js
└── bfti/
    └── index.html
```

## 本地预览

直接用浏览器打开 `index.html`。

## GitHub Pages

仓库名是 `martinaecc.github.io` 时，推送到 `main` 分支后通常会自动部署到：

```text
https://martinaecc.github.io/
```
